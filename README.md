# 🚀 Gestion Surcharge Performance

Système de monitoring et d'optimisation des performances avec Flask, Prometheus et Grafana.

## 🏗️ Architecture
```
Client → NGINX → Flask (x3) → Prometheus → Grafana
```

## 🛠️ Technologies

| Technologie | Version |
|-------------|---------|
| Flask | 2.x |
| NGINX | Load Balancer |
| Prometheus | Monitoring |
| Grafana | Dashboard |
| Docker Compose | Orchestration |

## 🚀 Installation
```bash
git clone https://github.com/Meima-Ely/gestion-surcharge-performance.git
cd gestion-surcharge-performance/backend
docker-compose up -d
```
